# RadioAccordionVue

This projet is a first contact with Storybook through Vue3 and Vite

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```

### Run Storybook

```sh
npm run storybook
```
